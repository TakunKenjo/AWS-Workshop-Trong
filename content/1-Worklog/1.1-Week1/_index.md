### Week 1 Objectives:

* Get familiar with the internship environment and FCJ team members.
* Learn the foundational knowledge of AWS Cloud, core services, and AWS management tools.
* Practice creating an AWS account, configuring MFA, managing costs with AWS Budgets, and deploying basic services.
* Gain in-depth knowledge of IAM, VPC, and EC2, and practice building a secure network infrastructure on AWS.
* Discuss as a team, select the "Smart Docs AI" project topic, and research ReactJS, Python, and the AWS ecosystem for project development.
* Design and develop the initial frontend of the project, push the source code to GitHub, and get hands-on experience with AWS Cloud9.

### Week 1 Achievements:

* Successfully bonded, interacted, and established an agile workflow with members of the First Cloud Journey (FCJ) team.
* Mastered the core fundamentals of AWS Cloud, gaining a solid understanding of global infrastructure concepts (Regions, AZs) and standard service categorizations.
* Successfully registered an AWS Free Tier account, maximized security posture using MFA, and configured active AWS Budgets to mitigate financial risks.
* Comprehended the granular authorization mechanisms of IAM (Users, Groups, Roles, Policies) and successfully provisioned an isolated, secure VPC network infrastructure.
* Finalized the project selection for **"Smart Docs AI"**, aligning the tech stack (ReactJS, Python, AWS), completing the initial UI blueprint, and structuring the codebase repository on GitHub.
* Got accustomed to and efficiently utilized cloud-native development environments (AWS Cloud9) as well as the Hugo documentation engine via Markdown.
* Learned how to effectively manage AWS expenditures, optimize AWS Free Tier quotas, and map out a structured learning pathway for advanced cloud services in the upcoming phase.
